Web Tools

The following is a list of web-tools from the projects I worked on at Lancaster University.

1- CFIE-FRSE-Desktop

Corporate Financial Information Environment (CFIE)-Final Report Structure Extractor (FRSE) is a desktop application to detect the structure of UK Annual Report and extract the reports' contents on a section level. CFIE-FRSE [https://drelhaj.github.io/CFIE-FRSE/]

The tool works as a desktop version of our CFIE-FRSE-Web tool (tool # 2 below).

2- CFIE-FRSE-Web

A Web-tool to Analyse UK Annual Report and Financial Documents. The tool is free for researchers please contact me to ask for log-in details. To access the web-tool:
CFIE WMATRIX-Import [https://cfie.lancaster.ac.uk:8443/]

For more details about the tools and to watch video tutorials please visit the link below:
CFIE WMATRIX-Import Details and Tutorials [http://ucrel.lancs.ac.uk/cfie/]

3- OSMAN Readability Metric

Java open Source tool for Arabic text readability. The tool calculates readability for Arabic text with and without diacritics (Tashkeel). The tool works better with diacritics added in (we provide a method to allow you add diacritics to plain Arabic text).
OSMAN Readability [http://drelhaj.github.io/OsmanReadability/]

4- Java word cloud with Log Likelihood

Java tool to create word clouds using log Likelihood and word frequencies (or any other weight values). Log Likelihood is calculated for a word between two large corpora input that could be in any language. The tool is language independent and was tested on Arabic and English
Java word cloud with Log Likelihood [http://drelhaj.github.io/Java_WordCloud_LogLikelihood/]

5- Machine Learning Java code

Java Code that trains classifiers for chairman's statements, governance & remuneration sections from 1,000 annual financial reports (Part of UCREL NLP Summer School 2016 Lancaster University)
Machine Learning Tutorial [https://github.com/drelhaj/MachineLearning/]

6- VardSourcing

A Web-tool to help participants to register to upload/VARD/download samples from EEBO Corpus.
VardSourcing [http://ucrel-vardsourcing.lancs.ac.uk/]
